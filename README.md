A custom CString class implemented in C++ without using the standard <string> library. This project demonstrates Object-Oriented Programming (OOP), dynamic memory allocation, operator overloading, and string manipulation techniques.

Features
Dynamic memory management
Copy constructor and destructor
Operator overloading (=, +, +=, [], <<)
String searching (find)
String insertion and removal
String replacement
Case conversion (makeUpper, makeLower)
String reversal
Trimming functions
Substring extraction (left, right)
String comparison (isEqual)
Concepts Used
Classes and Objects
Dynamic Memory Allocation
Deep Copying
Operator Overloading
Friend Functions
